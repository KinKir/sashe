Композиција на функции

24.11.2018, Колоквиум I, Група 2, Задача 1 Да се определи доменот на `f @ g` за :
а) `f(x) = x + 1/(x^2 -2x -3), g(x) = (3 -x)/x`
Решение

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in D_g | g(x) in D_f}`

Функцијата `f(x)` е дефинарана ако за именителот е `x^2 -2x -3 != 0`

`= x_(1,2) = (2 +- sqrt(4 + 12))/2 => x_1 = 3, x_2 = -1`

Доменот на функцијата `f` е `D_f = (-oo, -1] uu [-1, 3] uu [3, +oo)`

Доменот на функцијата `g` е `D_g = RR \\ {0}`

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in RR\\{0} | g(x) in (-oo, -1] uu [-1, 3] uu [3, +oo)}`

Излезите на функцијата `g(x)` треба да се во доменот `D_f`.

`(3 - x)/x != -1 ` и `(3 - x)/x != 3 => x = 3/4`

`D_(f@g) = (-oo, 0] uu [3/4, +oo)`

`■`

Задача 1 Да се одреди доменот и формулата за `f@g`:
а) `f(x) = (x + 1)/(x^2 + x - 6); g(x) = sqrt(x^2 - 1)`
Решение

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in D_g | g(x) in D_f}`

Функцијата `f(x)` не е дефинирана ако именителот е `0`, односно

`x^2 + x - 6 = 0`

` x_(1,2) = (-1 +- sqrt(1^2 -4(-6)))/2`

`x_1 = 2; x_2 = -3`

Доменот на функцијата `f` е `D_f = (-oo, -3) uu (2, +oo)`

Функцијата `g(x)` е дефинирана ако `x^2 - 1 >= 0`

`x_1 = 1; x_2 = -1`

Доменот на функцијата `g` е `D_g = (-oo,-1]uu[1, +oo)`

`D_(f@g) = (-oo, -3] uu [-3, -1) uu (1, 2] uu [2, +oo)`

Композицијата е: `(f @ g)(x) = f(g(x)) = (sqrt(x^2 - 1) + 1)/((sqrt(x^2 - 1))^2 + sqrt(x^2 - 1) - 6)`

`= (sqrt(x^2 - 1) + 1)/(sqrt(x^2 - 1) + x^2 - 7)`

`■`
