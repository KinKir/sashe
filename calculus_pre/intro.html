Припрема

Вежба 1. Нека е дадено:
`1/2 + 3/4 - 5/6`

Решение

`= (1*6)/(2*6) + (3*3)/(4*3) - (5*2)/(6*2)`
`= 6/12 + 9/12 - 10/12`
`= 6/12 + 9/12 - 10/12`
`= (6 + 9 - 10)/12`
`= 5/12`

`■`

Вежба 2.
`11/3 + 3/5 + 17/20`

Решение

`= (11*6)/(3*20) + (3*3)/(5*12) + (17*3)/(3*20)`
`= 220/60 + 36/60 + 51/60`
`= (220 + 36 + 51)/60`
`= 307/60`

`■`

Вежба 3.
`(5/6)/(15/22)`

Решение

`= (5/6)(22/15)`
`= (1/3)(11/3)`
`= 11/9`

Вежба 4.
`5/(xy) + 7/x^2 + (10x)/y`

Решение

`= (5*x)/((xy)*x) + (7*y)/(x^2*y) + (10x * x^2)/(y*x^2)`
`= (5x + 7y + 10x^3)/(x^2y)`

Вежба 5.
`((x^2 - 1)/(xy^2))(y^3/(x - 1))`

Решение

`= (((x - 1)(x + 1))/(xy^2))(y^3/(x - 1))`
`= ((x - 1)y)/x`
