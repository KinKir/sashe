Пропорции

Количникот на величините `a` и `b`, кога `b != 0`, е размер, а бројот `a : b` односно `a/b` е вредност на размерот. Ако размерите `a : b` и `c : d` имаат иста вредност, тогаш велиме дека сочинуваат пропорција.

`a:b = c:d`

Основни особини на пропорциите

`a:b = c:d <=> a = b * c/d`

`a:b = c:d <=> a:c= b:d`

Равенката `ax = b` каде `a in ZZ\\{0}, b in ZZ`, нема секогаш решение во множеството `ZZ` па се создава потреба за проширување на множеството воведувајки ги рационалните броеви. Множествот на рационалните броеви се означува со `QQ` и се дефинира како `QQ = {m/n | m in ZZ , n in NN}`. Цел број може да се претстави како дропка `m/1` па следи дека рационалните броеви ги содржат целите броеви `ZZ` а како овие ги содржат природните броеви `NN`, односно може да се запише `NN sub ZZ sub QQ`.

Во множеството на рационалните броеви `QQ` сите четри основни операции за пресметки се можни без ограничување освен делењето со нула.

За рационалниот број `q` велиме дека е децимална дропка ако може да се запише во облик:
`q = ±(a_0 + a_1 * 10^-1 + a_2*10^-2 + · · · + a_m*10^-m)`,
каде:
`a_0 = 0` или `a_0 ∈ NN` , `a_i ∈ {0, 1, . . . , 9}`, `i = 1, . . . , m`,
и го запишуваме во облик:
`q = ±a_0a_1a_2 · · · a_m`.
Такви се следниве децимални дропки:
`1962/1000 = 1 + 9/10 + 6/10^2 + 2/10^3 = 1.962` ;
`1994/100 = 19 + 9/10 + 4/10^2 = 19.94`

Ако дропката `a/b` не може да се запише како децимална дропка, односно ако делењето не завршува а при тоа одредена група на цифри постојано се повторува, велиме дека станува збор за периодична децимална дропка.
Бројот `7/22 = 0.31818181818 . . . ` е пример за периодична децимална дропка. Секоја периодична децимална дропка е воедно рационален број.

Ирационални броеви

Должината на дијагоналата `sqrt 2` на квадратот со страна `1` ни укажуваат дека постојат броеви кои не се рационални. Тоа се ирационалните броеви.

Ирационалните броеви можеме да ги дефинираме како непериодски децимални дропки со бескрајно многу децимали. Множеството на ирационалните броеви го означуваме со `bbbI`. Броевите `sqrt 2 = 1.414213562...`, и `pi = 3.14159265...` се примери за ирационални броеви.

Реални броеви

Рационалните и ирационалните броеви со едно име ги нарекуваме реални броеви и ги означуваме со `RR`. Значи `RR = QQ uu bbbI`. Покрај тоа `bbbI nn QQ = O/` и `NN sub ZZ sub QQ sub RR`.

Во множеството на реаните броеви дефенирани се операциите на собирање и множење, односно за секои два реални броја `x` и `y` еднозначно се одреддени реалните броеви `x + y` и `x * y`. Тие операции ги имаат следните својства:

  1. За сите `x, y, z in RR` важи `(x + y) + z = x + (y + z)` (закон за асоцијативно собирање)
  2. Постои само еден реален број `0 in RR` таков да за секој `x важи `x + 0 = 0 + x = x` (закон за асоцијативно собирање)
  3. За секој `x in RR` постои само еден `−x in RR` (инверзен елемент во поглед на собирање) таков да е `x + (−x) = (−x) + x = 0`.
  4. За сите `x, y in RR` важи `x + y = y + x` (закон за комутација за собирање).
  5. За сите `x, y, z in RR` важи `(xy)z = x(yz)` (закон за асоцијација за множење).
  6. Постои само еден реален број `1 in RR` (неутрален елемент за множење) таков да за секој `x in RR` важи `x * 1 = 1 * x = x`.
  7. За секој `x in RR, x != 0`, постои единствен елемент `x^−1 = 1/x in RR` (инверзен елемент за множење) таков да `xx^−1 = x^−1x = 1`.
  8. За сите `x, y in RR` важи `xy = yx` (закон на комутација за множење).
  9. За сите `x, y, z in RR` важи `x(y + z) = xy + xz` (закон за дистрибуција на множење спрема собирање)
  10. За било кои два реални броја `x, y in RR` важи `x <= y` или `y <= x`.
  11. `x <= y` и `y <= x`, тогаш и само тогаш кога `x = y`.
  12. Ако `x <= y` и `y <= z`, тогаш е `x <= z`.
  13. Ако `(x <= y)`, тогаш за секој `z in RR` важи `(x + z <= y + z)`
  14. `(0 <= x` и `0 <= y) -> (0 <= xy)`.

Интервали

Најчесто употребувани множества во математичката анализа се интервалите. Тоа се множества на реални броеви кои имаат својство да нивните елементи задоволуваат одредена нееднаквост.

  • Отворен интервал на реалните броеви `( a, b )`, одреден од два реални броја `a, b, a < b`, e множество на сите `x in RR` за кои важи `a < x < b`, односно `( a, b ) = {x in RR | a < x < b}`.
  • Затворен интервал или сегмент на реалните броеви `[a, b]`, одреден од два реални број `a, b, a <= b`, е множество од сите `x in RR` за кои важи `a <= x <= b`, односно `[a, b] = {x in RR | a <= x <= b}`.

Покрај отворените и затворените интервали дефинирани се и следните множества:

  • `(a, b] = {x in RR | a < x <= b} , [a, b) = {x in RR | a <= x < b}` ,
  • `(-oo, a ) = {x in RR | x < a} , (-oo, a] = {x in RR | x <= a}`,
  • `(a,oo) = {x in RR | x > a} , [a, oo) = {x in RR | x >= a}`.

Околина

Отворена околина на реалниот број `a` е секој отворен интервал од реални броеви кои го соджат бројот `a`.

Симетрична околина на реалниот број `a` е околина каде бројот `a` e sredina.

Апсолутна вредност на реалниот број

Ако `x` е реален број тогаш еден од броевите `x`, `−x` е ненегативен и го нарекуваме апсолутна вредност од `x`.

Во множеството на реаните броеви дефенирани се операциите на собирање и множење, односно за секои два реални броја `x` и `y` еднозначно се одреддени реалните броеви `x + y` и `x * y`. Тие операции ги имаат следните својства:

Асолутната вредност на реалниот број `x`, ја означуваме `abs(x)` и ја дефинираме како:

`abs(x) = {(x, x>=),(-x, x < 0):}`

Основни особини

`abs(x) >= 0`

`abs(x) = 0 <=> x = 0`

`abs(-x) = abs(x)`

`abs(-x)^2 = x^2`

`-abs(x) <= x <= abs(x)`

`abs(x) <= a <=> -a <= x <= a`

`abs(x) <= abs(y) <=> x^2 <= y^2`

`abs(abs(x) - abs(y)) <= abs(x - y)`

`abs(xy) = abs(x)abs(y)`

`abs(x/y) = abs(x)/abs(y)`

`abs(x + y) <= abs(x) + abs(y)`

`abs(x - y) <= abs(x) - abs(y)`

Степен на реален број

Степен на реален број `a^m; a in RR; m in NN`се дефинира

`a^0 = 1; a^1 = a; a^m = aa^(m-1); a != 0`

Основни особини (`a != 0; b != 0`)

`a^m a^n = a^(m+n)`

`a^m / a^n = a^(m-n)`

`(a^m)^n = a^(mn)`

`a^-m = 1/a^m`

`(ab)^m = a^mb^m`

`(a/b)^m = a^m/b^m`

`(0)^m = 0`

`0^0` не е дефинирано

Корен на реален број

Нека `x,y in RR^+` и `n,m in NN`. `n`-ти корен на бројот `x` е единственото позитивно решение на равенката `t^n = x` и се означува како `x^(1/n)` или `sqrt(x)`.

Основни особини

`root(n)(x^m) = x^(n/m)`

`root(n)(x^m) = x^(m/n) = (x^(1/n))^m = (root(n)x)^m`

`1/root(n)(x^m) = (1/x^(1/n))^m = 1/(x^(1/n))^m = x^(-m/n)`

`root(n) 0 = 0`

`root(n)x root(n)y = root(n)(xy)`

`root(nm)x = x^(1/(mn)`

`root(m)(root(n)x) = root(n)(root(m)x`

`root(n)1 = 1`

`a in RR => root(n)a^n = {(a, непарен),(abs(a), парен):}`
