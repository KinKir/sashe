The main objective of this section is to obtain formulas for the derivatives of the six basic trigonometric functions.

Ќе претпоставиме дека променливата `x` во тригонометриските функции `sinx`, `cosx`, `tanx`, `ctgx` е дадена во радијани. Исто така, ќе ги користиме следните два (претходно докажани) лимеси од тригонометриски функции во кои променливата `x` е заменета со променлива `h`:

`lim_(h->0) sinh/h = 1`, `lim_(h->0) (1 - cosh/h) = 1`

Извод на `f(x) = sinx`

` = lim_(h->0) (sin(x + h) - sin(x))/h`

` = lim_(h->0) (sinxcosh + cosxsinh - sinx)/h`

` = lim_(h->0) (sinx(cosh - 1) + cosxsinh)/h`

` = lim_(h->0) (sinx(cosh - 1))/h + (cosxsinh)/h`

` = lim_(h->0) sinx * lim_(h->0)(cosh - 1)/h + lim_(h->0)cosx * lim_(h->0) sinh/h`

` = lim_(h->0) sinx * 0 + lim_(h->0)cosx * 1`

` = cosx`

Во претпоследното равенство, се користи дека `sinx` и `cosx` не го вклучуваат `h`, па тие се константи кога `h -> 0`. Според тоа,

`lim_(h->0) sinx = sinx`, `lim_(h->0) cosx = cosx`

Извод на f (x) = cos x

Нека `f (x) = tanx`. Од правилото за извод на количник и претходно изведените изводи, имаме

` f(x) = lim_(h->0) (cos(x + h) - cos(x))/h`

` = lim_(h->0) (cosxcosh + sinxsinh - cosx)/h`

` = lim_(h->0) (cosx(cosh - 1) - sinxsinh)/h`

` = lim_(h->0) (cosx(cosh - 1))/h - (sinxsinh)/h`

` = lim_(h->0) cosx * lim_(h->0)(cosh - 1)/h - lim_(h->0)sinx * lim_(h->0) sinh/h`

` = lim_(h->0) cosx * 0 - lim_(h->0)sinx * 1`

` = -sinx`

Извод на `f (x) = tanx`

` f(x) = (sinx/cosx)' = lim_(h->0) ((sinx)'cosx - sinx(cosx)')/(cosx)^2`

` = (cosxcosx + sinxsinx)/(cosx)^2`

` = (cos^2x + sin^2x)/cos^2x`

`= 1/cos^2x`

Извод на `f(x) = ctgx`

Нека `f (x) = ctgx`. Од правилото за извод на количник и претходно изведените изводи, имаме

` f(x) = (cosx/sinx)' = lim_(h->0) (sinx(cosx)' - (sinx)'cosx)/(sinx)^2`

` = (-sinxsinx - cosxcosx)/(sinx)^2`

` = (-sin^2x - cos^2x)/sin^2x`

`= -1/sin^2x`

Извод на `f(x) = secx`

` f(x) = (secx)' = (1/cosx)'`

` = lim_(0h->0)((1)'cosx - 1*(cosx)')/cos^2x`

` = lim_(0h->0) (0 * cosx - 1* -(sinx))/cos^2x`

` = lim_(0h->0) sinx/cos^2x`

` = sinx/cos^2x = (sinx/cosx) * 1/cosx = tanxsecx`

Извод на `f(x) = cscx`

` f(x) = (secx)' = (1/sinx)'`

` = lim_(0h->0)((1)'sinx - 1*(sinx)')/sin^2x`

` = lim_(0h->0) (0 * sinx - 1* cosx)/sin^2x`

` = lim_(0h->0) -cosx/sin^2x`

` = sinx/cos^2x = (cosx/sinx) * 1/sinx = -cotxsecx`

Пример1. Да се најде `f'(x)`, ако `f(x) = x^2 tanx`.

`f'(x) = (x^2 tanx)^`

`= (x^2)'tanx * x^2(tanx)^'`

`= 2xtanx * x^2(1/cos^2x)^'`

Пример2. Да се најде `f'(x)`, ако `f(x) = sinx/(1 + cosx)`.

`f'(x) = (sinx/(1 + cosx))^'`

`= ((sinx)^'(1 + cosx) -sinx(1 + cosx)^')/(1 + cosx)^2`

`= (cosx(1 + cosx) - sinx(-sinx))/(1 + cosx)^2`

`= (cosx + cos^2x + sin^2x)/(1 + cosx)^2`

`= (1 +cosx)/(1 + cosx)^2`

`= 1/(1 + cosx)`
