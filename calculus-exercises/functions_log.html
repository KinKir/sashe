Вежба 1. Најди ги точните вредности без користење на калкулатор:
а) `log_2 (1/32)`

Решение

`log_2 (1/32) = x`

`2^x = 1/32`

`2^x = 32^-1`

`2^x =(2^5)^-1`

`2^x = 2^-5`

`x = -5`

`■`

б) `ln sqrt(e)`.
Решение

`e^x = sqrt(e)`

`e^x = e^(1/2)`

`x = 1/2`

`■`

Вежба 2. Изрази го логаритамот во термини на суми, разлики и производи од попрости логаритми
`ln sqrt((x^2 +1 )/(x^3 + 5))`

Решение

`= 1/2 ln (x^2 +1 )/(x^3 + 5)`

`= 1/2[ln (x^2 +1) - ln (x^3 + 5)]`

`■`

Вежба 3. Реши ги равенките:
а) `log_10 x^(3/2) - log_10 sqrt(x) = 5`

Решение

`3/2 log_10 x -1/2 log_10 x = 5`

`log_10 x = 5`

`x = 10^5`

`■`

б) `xe^-x + 2e^-x = 0`
Решение

`(x + 2)/e^x = 0`

`е^x > 0 => x + 2 = 0`

`x = -2`

`■`

в) `2e^(3x) = 7`
Решение

`ln (2e^(3x)) = ln 7`

`ln 2 + ln е^(3x) = ln 7`

`3x ln e^x = ln 7 - ln 2`

`x = 1/3 ln (7/2)`

`■`

Вежба 4. Скицирај го графикот:
а) `y = 3 + e^(x - 2)`

Решение
`y = e^x`
Со додавање на негативна константа `c` на `x`
функцијате е : `y = е^(x - 2)`
Со ова се врши поместување на графикот по `x` оската на десно за вредноста на негативната константа `c = 2`
`■`
Со додавање на позитивна константа `c` на `f(x)`
функцијате е : `y = 3 + е^(x - 2)`
Со ова се врши поместување на графикот по `y` оската на горе за вредноста на позитивната константа `c = 3`
`■`
б) `y = 3 ln root(3)(x - 1)`
Решение
`y = 3/3 ln (x -1)`
Прво ја цртаме `y = ln x`
Со додавање на негативна константа `c` на `x`
функцијате е : `y = ln (x - 1)`
Со ова се врши поместување на графикот по `x` оската на десно за вредноста на негативната константа `c = 1`
`■`

Што треба да се знае на памет
  • `log_b a = x` е идентично на `b^x = a`
  • `log_bxy = log_b x + log_b y`
  • `log_b(x/y) = log_bx - logby`
  • `log_bx^n = nlog_bx`
  • `log_b b = 1`
  • `log_b 1 = 0`
