Аритметички операции со функции

Вежба 1. Да се одредат домените на функциите `f + g`, `fg` и `f/g` ако:
а) `f(x) = 2/(x - 1)`, `g(x) = x-1`

Решение

`D_f = RR \ {1}`, `D_g = RR`

`D_(f + g) = D_f nn D_g = RR \\ {1}`

`D_(fg) = D_f nn D_g = RR \\ {1}`

`D_(f/g) = D_f nn D_g nn {x in D_g | g(x) != 0}`

`D_(f/g) = D_f nn D_g nn {x in RR | x-1 != 0} = RR \\ {1}`

`■`

б) `f(x) = sqrt (1 - x^2), g(x) = sqrt (2 + x - x^2)`
Решение

`D_f = [-1, 1], D_g = [-1, 2]`

`D_(f+g) = D_f nn D_g = `

`-x^2 + x + 2 >= 0`

`x_(1,2)= (-1+- sqrt(1 + 8))/ (-2) <=> x_1 = -1, x_2 = 2`

`D_(f + g) = D_f nn D_g = [-1, 1]`

`D_(fg) = D_f nn D_g = [-1, 1]`

`D_(fg) = D_f nn D_g nn {x in D_g | g(x) != 0}`

`D_(fg) = D_f nn D_g nn {x in RR | sqrt(2+ x - x^2) != 0}`

`= [-1, 1] nn {[-1, 2] \ {-1, 2} = (-1, 1]`

`■`

Композиција на функции

Вежба 1. Да се изрази функцијата `F` како композиција од три функции:
а) `F(x) = (1 + sin(x^2))^3`

Решение

`f_1(x) = x^2`

`f_2(x) = sin(x) `

`f_3(x) = x^3`

`F(x) = (f_3 @ f_2 @ f_1)(x)`

`■`

б) `F(x) = sqrt(1 - root(3)(x))`
Решение

`f_1(x) = root(3)(x)`

`f_2(x) = 1- x `

`f_3(x) = sqrt(x)`

`F(x) = (f_3 @ f_2 @ f_1)(x)`

`■`

Вежба 2. Да се определи доменот и правилото за `f @ g` и `g @ f` ако:
а) `f(x) = sqrt(x), g(x) = x^2 -5x + 6`
Решение

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in D_g | g(x) in D_f}`

Доменот на функцијата `f` е `D_f = [0, +oo)`

Доменот на функцијата `g` е `D_g = RR`

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in RR | g(x) in [0, +oo)}`

Излезите на функцијата `g(x)` треба да се во доменот `D_f`.

`x^2 -5x + 6 >= 0 => x^2 -5x + 6 = 0`

`= x_(1,2) = (5 +- sqrt(25 -24))/2`

`x_1 = 3, x_2 = 2`

`=> x^2 -5x +6 >= 0, x in (-oo, 2] uu [3, +oo)`

`D_(f@g) = (-oo, 2] uu [3, +oo)`

Композицијата е: `(f @ g)(x) = f(g(x)) = sqrt(x^2 -5x +6)`

-----------------------------------------------------------------------

Доменот на композицијата `g @ f` е `D_(g @ f) = {x in D_f | f(x) in D_g}`

Доменот на функцијата `f` е `D_f = [0, +oo)`

Доменот на функцијата `g` е `D_g = RR`

Доменот на композицијата `g @ f` е `D_(g @ f) = {x in [0, +oo) | f(x) in RR}`

Излезите од функцијата `f(x)` треба да се во доменот `D_g`.

`sqrt(x) = 0`

`x = 0 => sqrt(x) >= 0 x in [0, +oo)`

`D_(g@f) = [0, +oo)`

Композицијата е: `(g @ f)(x) = g(f(x)) = x + 5sqrt(x) +6)`

`■`

б) `f(x) = sqrt(9 - x^2), g(x) = |~ x ~|` (најмалиот цел број `>= x`)
Решение

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in D_g | g(x) in D_f}`

Доменот на функцијата `f` е `D_f = [-3, 3]`

Доменот на функцијата `g` е `D_g = RR` (зошто `RR` пр. `x = - 2.4 => g(x) = 2`)

`f @ g` е `D_(f @ g) = {x in RR | g(x) in [-3, 3]} = [-3, 3]`

`f@g = f(g(x)) = sqrt(9- |~x~|^2)`

----------------------------------------------------------------------------------

Доменот на композицијата `g @ f` е `D_(g @ f) = {x in D_f | f(x) in D_g}`

Доменот на функцијата `f` е `D_f = [-3, 3]`

Доменот на функцијата `g` е `D_g = RR`

`g(x) = 3, x in (-4, -3) uu (2,3)`

`f @ g` е `D_(f @ g) = {x in [-3, 3] | g(x) in RR} = (-4, 3]`

`g@f = g(f(x)) = |~sqrt(9 - x^2)~|`

`■`

в) `f(x) = 1/(x - 1), g(x) = 1/(x +1)`
Решение

Доменот на композицијата `f @ g` е `D_(f @ g) = {x in D_g | g(x) in D_f}`

Доменот на функцијата `f` е `D_f = RR \\{1}`

Доменот на функцијата `g` е `D_g = RR \\{-1}`

Доменот на `f @ g` е `D_(f @ g) = {x in RR \\ {-1} | g(x) in {1}`

Излезите на функцијата `g(x)` треба да припаѓаат на доменот `D_f`.

`1/(x + 1) = 1 => x = 0`

`f @ g` е `D_(f @ g) = {x in RR \\ {-1} | g(x) in {1} = RR\\{-1, 0}`

`f@g = f(g(x)) = 1/(1/(x + 1) - 1) = 1/((1- x - 1)/(x + 1)) = - (x + 1)/x`

----------------------------------------------------------------------------------

Доменот на композицијата `g @ f` е `D_(g @ f) = {x in D_f | f(x) in D_g}`

Доменот на функцијата `f` е `D_f = RR \\ {1}`

Доменот на функцијата `g` е `D_g = RR \\{-1}`

Доменот на `g @ f` е `D_(g @ f) = {x in RR \\ {1} | f(x) in RR \\ {-1}`

Излезите на функцијата `f(x)` треба да припаѓаат на доменот `D_g`.

`1/(x - 1) != -1 => x= 0`.

`D_(g @ f) = {x in RR \\ {1} | f(x) in {-1} = RR \\{0, 1}`

`g @ f = g(f(x)) = 1/(x + 1) = 1/(1/(x-1) + 1) = (x - 1)/1`

`■`

График на функција

Вежба 1. Да се скицира графикот на функцијата:
а) `y = 2- 1/(x + 1)` користејки го графикот `y = 1/x`

Решение
Чекор 1: `y = 1/x`
Чекор 2: Со додавање на позитвна константа `c` на `x`, `f(x)`
функцијате е : `y = f(x + 1) = 1/(x +1)`
Со ова се врши поместување на графикот по `x` оската налево за вредноста на позитивната константа `c = 1`
Чекор 3: Со множење на функцијата `f(x)` со `-1`
функцијате е : `y = -f(x) = -(1/(x +1))`
со што се добива рефлекција по `x` оската.
Чекор 4: Со додавање на позитивна констанра `c` на функцијата `f(x)`
функцијате е : `y = f(x) + 2 = 2 - (1/(x +1))`
со што се добива поместување нагоре по `y` - оската.

`■`

б) `y = abs(2x - 1) + 1`, користејки `y = abs(x)`
Решение
Чекор 1: `y = abs(x)`
Чекор 2: Со множење на функцијата со константа `c, c > 0`,
функцијата е: `y = cf(x) = abs(2x)`
со што се добива вертикално истегање со фактор `c`.
Чекор 3: Со вадење на позитивна константа `c`,
функцијата е: `y = f(x - c) = abs(2(x - 1/2))`
со што се добива поместување надесно по `x` оската за `c = 1/2`.
Чекор 4: Со додавање на позитивна константа `c` н афункцијата,
функцијата е: `y = f(x) + c = abs(2x - 1)) + 1`
со што се добива поместување нагоре по `y` оската за `c = 1`.

`■`

в) `y = - sqrt(3x)`, користејки `y = sqrt(x)`
Решение
Чекор 1: `y = sqrt(x)`
Чекор 2: Со множење со константа `c, c > 0`,
функцијата е: `y = cf(x) = sqrt(3)sqrt(x) = sqrt(3x)`
со што се добива вертикално истегање со фактор `c = sqrt(3)`.
Чекор 3: Со множење на функцијата со `-1`,
функцијата е: `y = -f(x) = - sqrt(3x)`
со што се добива рефлекција по `x` оската.

`■`

Парни и непарни функции

Вежба 1. Да се одреди парноста/непарноста на функциите
а) `f(x) = x^2`

Бидејки е `f(x) = f(-x)`, односно е симетрична во однос на `y` оската, функцијате е парна.
б) `f(x) = -x + 2`
Бидејки функцијата не е симетрична во однос на `y` оската ниту е симетрична во однос на кординатниот почеток, функцијате не е ниту парна ниту не-парна.

Вежба 2. Да се испита парноста на функциите:
а) `f(x) = (x^5 - x)/(1 + x^2)`

Решение

`f(-x) = (-x^5 - (-x))/((1 + (-x^2))`

`f(-x) = (-x^5 + x)/(1 + x^2)`

`f(-x) = -(x^5 + x)/(1 + x^2)`

`f(-x) = -f(x)`

Функцијата е непарна

`■`

б) `f(x) = 2`
Решение

`f(-x) = 2`

`f(-x) = f(x)`

Функцијата е парна

`■`

в) `f(x) = abs(x)/x`
Решение

`f(-x) = abs(-x)/-x`

`f(-x) = abs(x)/-x`

`f(-x) = -(abs(x)/x)`

`f(-x) = -f(x)`

Функцијата е непарна

`■`

г) `f(x) = (x - 2)^2`
Решение

`f(-x) = (-x - 2)^2`

`f(-x) = (x + 2)^2`

`f(-x) != -f(x) != f(x)`

Функцијата ниту е парна ниту е непарна

`■`

Што треба да се знае на памет
  • Дефиниција: За функциите `f + g`, `f – g`, и `fg` доменот е пресек на домените на `f` и `g` , a за функцијата `f/g` доменот e пресек на домените на `f` и `g` без оние точки `x` за кои `g(x) = 0`.
  • Дефиниција: Композиција `f@g` на функциите `f` и `g` се дефинира со:`(f@g)(x) = f(g(x))`. Доменот на `f@g` се состои од сите `x` од доменот на `g` за кои `g(x)` е во доменот на `f`, oдносно:
    `D_(f@g) = {x in D_g | g(x) in D_f}`
    Аналогно, композицијата `g@f` на функциите `g` и `f` се дефинира со:
    `(g@f)(x) = g(f(x))`
    Доменот на `g@f` се состои од сите `x` од доменот на `f` за кои `f(x)` е во доменот на `g`, oдносно:
    `D_(g@f) = {x in D_f | f(x) in D_g}`
  • Нови функции добиени од постојни
    Транслација:
    `y = f(x) + c`  транслација нагоре по `y` оската
    `y = f(x) - c`  транслација надоле по `y` оската
    `y = f(x + c)`  транслација налево по `x` оската
    `y = f(x - c)`  транслација надесно по `x` оската
    Рефлексија:
    `y = f(-x)`  рефлексија по `y` оската
    `y = -f(x)`  рефлексија по `x` оската
    `y = f(x + c)`  транслација налево по `x` оската
    `y = f(x - c)`  транслација надесно по `x` оската
    Растегање и збивање:
    `y = cf(x), c > 1`  растегање по `x` оската со фактор `c`
    `y = cf(x), 0 < c < 1`  збивање по `x` оската со фактор c
    `y = f(cx), c > 1`<  збивање по `x` оската со фактор c
    `y = f(cx), 0 < c < 1`  растегање по `x` оската со фактор `c`
  • Парни и непарни функции:
    Парна функција (симетрична во однос на `y` оската): `f(x) = f(-x), x in D_f`
    Непарна функција (симетрична во однос на кординатнот почеток):  `f(-x) = -f(x)`
