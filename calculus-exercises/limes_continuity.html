Непрекинатост на функција

Да се одреди дали функцијата е за дадена вредност на `a`. Ако е непрекината да се пресмета вредноста на функцијата за `a`.

Дефиниција. 1 Функција `f` е непрекината во точка `x = c`, ако се исполнети следниве услови:
  • 1. `f (x)` e дефинирана во `x=c`.
  • 2. `lim_(x->c)f (x)` постои.
  • 3. `lim_(x->c)f(x) = f(c)`

Вежба 1. `a = 2`
`f(x) = {(1/(x - 2), x !=2),(3, x = 2):}`

Решение

Услов 1: Функцијата да e дефинирана во `f(a)`
`f(a) = 3`

Услов 2: Да постои лимес за точката `a` `lim_(x->a)f(x)`
`lim_(x->2^-) 1/(x - 2)`
`= lim_(x->2^-) -1/0`
`= -oo`
`lim_(x->2^+) 1/(x - 2)`
`= lim_(x->2^-) +1/0`
`= +oo` Лимес не постои

Функцијата е прекината во точката `a`

Вежба 2. `a = 1`
`f(x) = {(1 +x^2, x <= 1),(4sqrt(x)-2, x> 1):}`

Решение

Услов 1: Функцијата да e дефинирана во `f(a)`
`f(a) = 1 + a^2 = 2`

Услов 2: Да постои лимес за точката `a` `lim_(x->a)f(x)`
`lim_(x->1^-) 1 + x^2`
`= lim_(x->1^-) 1 + 1`
`= 2`
`lim_(x->1^+) 4sqrt(x)-2`
`= lim_(x->2^-) 4*1 -2`
`= 2` Бидејки `lim_(x->a^-)f(x) = lim_(x->a^+)`, лимес lim_(x->a) f(x)`постои.

Услов 3: `lim_(x->a)f(x) = f(a)`
`lim_(x->1)f(x) = f(1) = 2`

Функцијата е непрекината во точката `a = 1`

Вежба 3. `a = 0` и `a = pi`
`f(x) = {(2+ x^2, x <= 0),(2cosx, 0 < x <=pi), (sinx -2, pi < x):}`

Решение

`a = 0`

Услов 1: Функцијата да e дефинирана во `f(a)`
`f(a) = 2 + a^2 = 2cos(a) = 2`

`a = 0`

Услов 2: Да постои лимес за точката `a` `lim_(x->a)f(x)`
`lim_(x->0^-) 2 + x^2`
`= lim_(x->0^-) 2 + 0^2`
`= 2`
`lim_(x->0^+) 2cosx`
`= lim_(x->0^+) 2*1`
`= 2`
Бидејки `lim_(x->a^-)f(x) = lim_(x->a^+)`, лимес lim_(x->a) f(x)`постои.

Услов 3: `lim_(x->a)f(x) = f(a)`
`lim_(x->1)f(x) = f(0) = 2`

Функцијата е непрекината во точката `a = 0`

`a = pi`

Услов 1: Функцијата да e дефинирана во `f(a)`
`f(a) = 2cos(pi) = sin(pi) -2 = -2`

`a = 0`

Услов 2: Да постои лимес за точката `a` `lim_(x->a)f(x)`
`lim_(x->pi^-) 2cosx`
`= lim_(x->0^-) 2 * -1`
`= 2`
`lim_(x->0^+) sinx -2`
`= lim_(x->0^-) 0 - 2`
`= -2` Бидејки `lim_(x->a^-)f(x) = lim_(x->a^+)`, лимес lim_(x->a) f(x)`постои.

Услов 3: `lim_(x->a)f(x) = f(a)`
`lim_(x->pi)f(x) = f(0) = -2`

Функцијата е непрекината во точката `a = 0`

Да се одреди вредноста на `c` така да функцијата биде непрекината секаде.

Вежба 1.
`f(x) = {(cx - 2, x <= 2),(cx^2 + 1, 2 < x):}`

Решение

Пресметуваме лев и десен лимес: `lim_(x->2^-) cx - 2`
`= lim_(x->2^-)c*2 - 2`
`= lim_(x->2^-) 2c - 2`
--------------------------- `lim_(x->2^+) cx^2 + 2`
`= lim_(x->2^+)c*2^2 + 2`
`= lim_(x->2^+) 4c + 2`

За да функцијата биде непрекината насекаде во деменот на функцијата (ова е глуп гооглe транслате превод на elsewheare без да се води сметка за контекстот на реченицата. Предходно баравме непрекинатост на функција во одредена точка а сеgа - elsewhere - на друго место), трeба `lim_(x->a)f(x)=f(a)`, односно: `2c - 2 = 4c + 2`
`c = -3/2`

Дефиниција. 3 Функција `f` e непрекината од лево во точката `x = c`, ако

`lim_(x->c^-)f (x) = f(c)`

и е непрекината од десно во точката `x = c`, ако

`lim_(x->c^+)f (x) = f(c)`
