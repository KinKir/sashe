Лимеси во бескрајност

Вежба 1. Да се пресмета:
`lim_(x -> +oo) 1/(3x + 4)`

Решение

Броителот и именителот го делиме со најголемиот степен на `x`
`= lim_(x -> +oo) (1/x)/((3x)/x + 4/x)`
`= lim_(x -> +oo) (1/x)/(3 + 4/x)`
`= lim_(x -> +oo) (1/(+oo))/(3 + 4/(+oo))`
`= lim_(x -> +oo) 0/(3 + 0)`
`= 0`

Вежба 2. Да се пресмета:
`lim_(x -> +oo) [x^2(x + 1)(3 - x)]`

Решение

Треба да се изврши множење за да се добие полином:

`= lim_(x -> +oo) [-x^4 + 2x^3 + 3x^2]`

Однесувањето на полиномна функција во бесконечност е одредедено со однесувањето во бесконечност на нејзиниот член со највисок степен.

`=lim_(x -> +oo) (-x^4)`
`= -oo`

Вежба 3. Да се пресмета:
`lim_(x -> +oo) (3x +4)/(x-7)`

Решение

Броителот и именителот го делиме со најголемиот степен на `x`

`= lim_(x -> +oo) ((3x)/x + 4/x)/(x/x - 7/x)`
`= lim_(x -> +oo) (3 + 4/(+oo))/(1 - 7/(+oo))`
`= lim_(x -> +oo) (3 + 0)/(1 - 0)`
`= 3`

`=lim_(x -> +oo) (-x^4)`
`= -oo`

Вежба 4. Да се пресмета:
`lim_(x -> +oo) cosx`

Решение

Бидејки `cosx` не клони кон некоја одредена вредност кога `x` клони кон `+oo`, лимесот не постои.

Вежба 5. Да се пресмета:
`lim_(x -> +oo) (5x^4 + 5)/((x^2 - 1)(2x^3 + 3))`

Решение

Множиме во именителот за да добиеме полином.

`= lim_(x -> +oo) (5x^4 + 5)/(2x^5 - 2x^3 + 3x^2 - 3)`

Сега делиме со највисокиот степен на `x` во именителот:

`= lim_(x -> +oo) (5x^4/x^5 + 5/x^5)/(2x^5/x^5 - 2x^3/x^5 + 3x^2/x^5 - 3/x^5)`
`= lim_(x -> +oo) (5/x + 5/x^5)/(2 - 2/x^2 + 3/x^3 - 3/x^5)`
`= lim_(x -> +oo) (0 + 0)/(2 - 0 + 0 - 0)`
`= 0`

Вежба 6. Да се пресмета:
`lim_(x -> -oo) x/sqrt(x^2 - 1)`

Решение

Делиме со највисокиот степен на `x` во именителот:

`lim_(x -> -oo) (x/x)/(sqrt(x^2 - 1)/x)`

Бидејки `x` се стреми кон `-oo` треба да го означиме со знак `-`.

`lim_(x -> -oo) 1/(sqrt(x^2 - 1)/(-x)`
`lim_(x -> -oo) 1/-(sqrt(x^2/x^2 - 1/x^2)`
`lim_(x -> -oo) 1/-(sqrt(1 - 1/x^2)`
`lim_(x -> -oo) 1/-(sqrt(1 - 0)`
`= -1`

Вежба 7. Да се пресмета:
`lim_(x -> -oo) (8x^4 + 3x - 5)/(x^2 + 1)`

Решение

Делиме со највисокиот степен на `x` во именителот:

`= lim_(x -> -oo) ((8x^4)/x^2 + 3x/x^2 - 5/x^2)/(x^2/x^2 + 1/x^2)`
`= lim_(x -> -oo) (8x^2 + 3/x - 5/x^2)/(1 + 1/x^2)`
`= lim_(x -> -oo) (8x^2 + 0 - 0)/(1 + 0)`
`= +oo`

Вежба 8. Да се пресмета:
`lim_(x -> +oo) (sqrt(x) - x)`

Решение

Со зaмена се добива обликот `+oo - +oo`. Потребно е изразот да го упростиме:

`= lim_(x -> +oo) sqrt((x))(1 - sqrt(x))`
`= lim_(x -> +oo) sqrt(x) * lim_(x->-oo)(1- sqrt(x))`
`= lim_(x -> +oo) sqrt(x) * lim_(x->-oo)1 - lim_(x->+oo) sqrt(x))`
`= +oo * 1 -oo`
`= +oo * -oo`
`= -oo`
