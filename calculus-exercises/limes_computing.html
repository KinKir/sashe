Лимеси - пресметка

Вежба 1. Нека е дадено:
`lim_(x -> a) f(x) = 2; lim_(x -> a) g(x) = -4; lim_(x -> a) h(x) = 0`
да се најде лимесот
a. `lim_(x -> a) [f(x) + 2g(x)]`

Решение

Според Теоремата 2,
`lim_(x -> a) [f(x) + 2g(x)] = lim_(x -> a) f(x) + 2lim_(x -> a) g(x) = 2 + 2*(-4) = -6`

`■`

b. `lim_(x -> a) [h(x) - 3g(x) +1]`
Решение

Според Теоремата 2,
`lim_(x -> a) [h(x) - 3g(x) +1] = lim_(x -> a) h(x) - 3lim_(x -> a) g(x) + 1 = 0 - 3*(-4) + 1 = 13`

`■`

c. `lim_(x -> a) [f(x) * g(x)]`
Решение

Според Теоремата 2,
`lim_(x -> a) [f(x) * g(x)] = lim_(x -> a) f(x) * lim_(x -> a) g(x) = 2 * (-4) = -8`

`■`

d. `lim_(x -> a) root(3) [6 + f(x)]`
Решение

Според Теоремата 2,
`lim_(x -> a) root(3) [6 + f(x)] = root(3) (lim_(x -> a) (6 + f(x)) = root(3) 8 = 2`

`■`

e. `lim_(x -> a) 2/g(x)`
Решение

Според Теоремата 2,
`lim_(x -> a) 2/g(x) = (lim_(x->2) 2)/( lim_(x->2) g(x)) = 2/(-4)= -1/2`

`■`

Вежба 2. Со помош на графикот за `f` и `g` најди го лимесот. Ако не постои објасни зошто.

a.`lim_(x -> 2) [f(x) + g(x)]`

Решение

Функцијата `f(x)` нема лимес `lim_(x->2) f(x)`

`■`

b.`lim_(x -> 0) [f(x) + g(x)]`
Решение

Функцијата `f(x)` нема лимес `lim_(x->0) f(x)`

`■`

c.`lim_(x -> 0^+) [f(x) + g(x)]`
Решение

`lim_(x -> 0^+) [f(x) + g(x)] = -2 + 2 = 0`

`■`

d.`lim_(x -> 0^-) [f(x) + g(x)]`
Решение

`lim_(x -> 0^-) [f(x) + g(x)] = 1 + 2 = 3`

`■`

Вежба 3.`lim_(x -> 2) x(x-1)(x+1)`

Решение

Според Теоремата 2,
`lim_(x -> 2) x(x-1)(x+1) = lim_(x->2) 2 * lim_(x->2) (3 - 1) * lim_(x->2) (x+1) = 2*1*3 = 6`

`■`

Вежба 5.`lim_(x -> 3) (x^2 - 2x)/(x + 1)`

Решение

Според Теоремата 2,
`lim_(x -> 3) (x^2 - 2x)/(x + 1) = (lim_(x->3) (x^2-2x)) / (lim_(x->3) (x+1)) = (3^2 -2*3)/(3+1) = 3/4`

`■`

Вежба 7.`lim_(x -> 1^+) (x^4 - 1)/(x - 1)`

Решение

Прво треба да се упрости изразот во броителот
`lim_(x -> 1^+) (x^4 - 1)/(x - 1) = lim_(x -> 1^+) ((x-1)(x+1)(x^2+1))/(x - 1) = lim_(x -> 1^+) (x^3 +x^2+x+1) = 1^3+1^2+1+1=4`

`■`

Вежба 8.`lim_(x -> -2) (x^3 +8)/(x + 2)`

Решение

Разложување со примена на формулите за скратено множење `(x^3+-y^3)=(x+y)(x^2 +- xy + y^2)`
`lim_(x -> -2) (x^3 +8)/(x + 2) = lim_(x -> -2) (x^3 +2^3)/(x + 2) = lim_(x -> -2) ((x +2)(x^2 + 2x + 4))/(x + 2) = lim_(x -> -2) (x^2 + 2x + 4)=4`

`■`

Вежба 9.`lim_(x -> -1) (x^2 + 6x + 5)/(x^2 -3x - 4)`

Решение

Со замена се добива неопределениот облик `0/0`
Потребно е разложување на квадратен трином на линеарни множители `ax^2 + bx + c = a(x-x_1)(x-x_2)`
`= lim_(x -> -1) (1(x+1)(x+5))/(1(x-4)(x+1))`
`= lim_(x -> -1) (x+5)/(x-4)`
`=-4/5`

`■`

Вежба 10.`lim_(x -> 2) (x^2 -4x + 4)/(x^2 + x - 6)`

Решение

Со замена се добива неопределениот облик `0/0`
Потребно е разложување на квадратен трином на линеарни множители `ax^2 + bx + c = a(x-x_1)(x-x_2)`
`= lim_(x -> 2) (1(x-2)(x-2))/(1(x-2)(x+3))`
`= lim_(x -> 2) (x-2)/(x+4)`
`= 0/6 = 0`

`■`

Вежба 11.`lim_(x -> -1) (2x^2 +x - 1)/(x + 1)`

Решение

Со замена се добива неопределениот облик `0/0`
Потребно е разложување на квадратен трином на линеарни множители `ax^2 + bx + c = a(x-x_1)(x-x_2)`
`= lim_(x -> -1) (2(x-1)(x+1))/(x + 1)`
`= lim_(x -> -1) 2(x - 1)`
`= -4`
