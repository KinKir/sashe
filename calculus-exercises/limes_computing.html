Лимеси - пресметка

`lim_(x->a)k=k,`  `lim_(x->a^+)x=a,`  `lim_(x->o^-)1/x=-oo,`  `lim_(x->o^+)1/x=+oo,`

Вежба 1. Нека е дадено:
`lim_(x -> a) f(x) = 2; lim_(x -> a) g(x) = -4; lim_(x -> a) h(x) = 0`
да се најде лимесот
a. `lim_(x -> a) [f(x) + 2g(x)]`

Решение

Според Теоремата 2,
`lim_(x -> a) [f(x) + 2g(x)] = lim_(x -> a) f(x) + 2lim_(x -> a) g(x) = 2 + 2*(-4) = -6`

`■`

b. `lim_(x -> a) [h(x) - 3g(x) +1]`
Решение

Според Теоремата 2,
`lim_(x -> a) [h(x) - 3g(x) +1] = lim_(x -> a) h(x) - 3lim_(x -> a) g(x) + 1 = 0 - 3*(-4) + 1 = 13`

`■`

c. `lim_(x -> a) [f(x) * g(x)]`
Решение

Според Теоремата 2,
`lim_(x -> a) [f(x) * g(x)] = lim_(x -> a) f(x) * lim_(x -> a) g(x) = 2 * (-4) = -8`

`■`

d. `lim_(x -> a) root(3) [6 + f(x)]`
Решение

Според Теоремата 2,
`lim_(x -> a) root(3) [6 + f(x)]`
`= root(3) (lim_(x->a)(6+f(x)))`
`= root(3) 8 = 2`

`■`

e. `lim_(x -> a) 2/g(x)`
Решение

Според Теоремата 2,
`lim_(x -> a) 2/g(x) = (lim_(x->2) 2)/( lim_(x->2) g(x)) = 2/(-4)= -1/2`

`■`

Лимес од полиноми и рационални функции кога `x-> a`

Лимес од рационални функции кога `x-> a`

Лимес од полином `p(x)` и рационална функција е еднаков на вредноста на полиномот или рационалната функција за ведноста кон која клони променливата `x`. Пресметката на лимес на полином ја сведуваме на пресметка на полиномот или рационалната функција за дадената вредност на `x`.

Вежба 3.`lim_(x -> 2) x(x-1)(x+1)`

Решение

Според Теоремата 2,
`lim_(x -> 2) x(x-1)(x+1) = lim_(x->2) 2 * lim_(x->2) (2 - 1) * lim_(x->2) (2+1) = 2*1*3 = 6`

`■`

Вежба 4.`lim_(x -> 3) x^3 - 3x^2 + 9x`

Решение

Според Теоремата 2,
`lim_(x -> 3) x^3 - 3x^2 + 9x = = lim_(x-> 3) x^3 - 3lim_(x->3) x^2 +9lim_(x->3) x`
`=27 -27 + 27 = 27`

`■`

Вежба 5.`lim_(x -> 3) (x^2 - 2x)/(x + 1)`

Решение

`lim_(x -> 3) (x^2 - 2x)/(x + 1) = (lim_(x->3) (x^2-2x)) / (lim_(x->3) (x+1)) = (3^2 -2*3)/(3+1) = 3/4`

`■`

Вежба 6.`lim_(x -> 2^+) sqrt(2 - x)`

Решение

Кога `x` клони кон `2` од десно изразот `2-x е негативен и не е во доменот на функцијата, па лимесот не постои.

Неопределена вредност `k/0`

За рационални функции каде лимесот на именителот е нула не може да се одреди лимесот. Постојат два случаи кои треба да се разгледаат, случајот каде лимесот на именителот е нула и случајот каде лимесот и на именителот и на броителот е нула. Во случајот кога лимесот на именителот е нула но на броителот не е нула, може да се докаже дека лимес не постои и дека ја имаме една од следите ситуации:

  • Лимесот може да биде `-oo` од една страна и `+oo` од друга страна.
  • Лимесот може да биде `+oo`.
  • Лимесот може да биде `-oo`.

Вежба 1.`lim_(x -> 1^+) 3/ (x -1)`

Решение

Со дирекна замена го добиваме неопределениот облик `k/0`. Кога `x` клони кон `1` од десно, изразот `(x – 1)` е со позитивен предзнак [пр. 1.000001 - 1 > 0], така да

`lim_(x -> 1^+) 3/ (x -1)`
`= +oo`

Лимесот е позитивна бескрајност бидејки количникот на `3` со многу мала позитивна вредност блиска до `0` дава голем позитивен број.

`■`

Вежба 2a.`lim_(x -> 3^+) x/(x - 3)`

Решение

Со замена на вредноста за `x` се добива неопределениот облик `k/0`. За да го одредиме предзнакот земамае број малку поголем од `3`. Сега предзнакот во именителот е `+`, лимесот е :

`lim_(x -> 3^+) x/(x - 3) = + oo`

`■`

Вежба 2b.`lim_(x -> 3^-) x/(x - 3)`

Решение

Со замена на вредноста за `x` се добива неопределениот облик `k/0`. За да го одредиме предзнакот земамае број малку помал од `3`. Сега предзнакот во именителот е `-`, лимесот е :

`lim_(x -> 3^-) x/(x - 3) = - oo`

`■`

Вежба 2c.`lim_(x -> 3) x/(x - 3)`

Решение

Со замена на вредноста за `x` се добива неопределениот облик `k/0`. Лимес не постои.

`= -oo`

Неопределена вредност `0/0`

Со замена се добива неопределениот облик `0/0`, што значи дека броителот и именителот имаат заеднички множител `x – a`. Од овде следи дека може да се поделат со `x – a`. Во овој случај, прво се крати заедничкиот множител `x – a`, а потоа се бара лимесот на добиената поедноставна функција.

Вежба 1.`lim_(x -> 1^+) (x^4 - 1)/(x - 1)`

Решение

`= lim_(x -> 1^+) ((x-1)(x+1)(x^2+1))/(x - 1)`
`= lim_(x -> 1^+) (x^3 +x^2+x+1)`
`= 1^3+1^2+1+1=4`

`■`

Вежба 2.`lim_(x -> -2) (x^3 +8)/(x + 2)`

Решение

Кај броителот применуваме разложување со примена на формулите за скратено множење `(x^3+-y^3)=(x+y)(x^2 +- xy + y^2)`
`= lim_(x -> -2) ((x +2)(x^2 + 2x + 4))/(x + 2)`
` = lim_(x -> -2) (x^2 + 2x + 4)=4`

`■`

Вежба 3.`lim_(x -> -5) (x^2 + 5x)/(x^2 - 25)`

Решение

`= lim_(x -> -5) (x(x + 5))/((x - 5)(x +5))`
`= lim_(x -> -5) x/(x - 5)`
`= 1/2

Решение

`= lim_(x -> -1) (1(x+1)(x+5))/(1(x-4)(x+1))`
`= lim_(x -> -1) (x+5)/(x-4)`
`=-4/5`

Вежба 12.`lim_(x -> (4/3)) (3x^2 - 4x)/ abs(3x - 4)`

Решение

Дирекната замена дава неопределен облик `0/0`.

Треба да се одреди левиот и десниот лимес за да се одреди дали лимесите се еднакви. За да го најдеме левиот лимес, ќе ја разгледаме вредноста која е маклу помала од `4/3` и ќе ја вметнемево лимесот. За да се упрости апосутната вредност во именителот, ознаките за апсолутна вредност `|` се заменуваат со мали загради `(` и се додава негативен знак, бидејки замената со вредност помала од `4/3` ќе го направи бројот во заградата негативен а додадениот негативен знак ќе ја направи вредноста повторно позитивна.

`lim_(x -> (4/3)^-) (3x^2 - 4x)/ -(3x - 4)`
`lim_(x -> (4/3)^-) (x(3x - 4))/ -(3x - 4)`
`lim_(x -> (4/3)^-) -x`
`= -4/3`

Десниот лимес гоодредуваме слично. Сега знакот за аполутна вредност едноставно го заменуваме со мали загради, негативниот знак не е потребен бидејнки вредноста во заградите е позитивна кога `x` го менуваме со вредност малку поголема од `4/3`

`lim_(x -> (4/3)^-) (3x^2 - 4x)/ (3x - 4)`
`lim_(x -> (4/3)^-) (x(3x - 4))/ (3x - 4)`
`lim_(x -> (4/3)^-) x`
`= 4/3`

Бидејки левиот и десниот лимес не се еднакви, лимесот не постои.

Вежба 17.`lim_(x -> 3^+) abs(3 - x)/ (x^2 - 2x - 3)`
Решение

Дирекната замена дава неопределен облик `0/0`.

Кога кон `3` се стремиме од десно изразот `abs(3 - x)` станува `-(3 - x), и со упростување на именителот добиваме:

`= lim_(x -> 3^+) -(3 - x)/ ((x - 3)(x + 1))`
`= lim_(x -> 3^+) (x - 3)/ ((x - 3)(x + 1))`
`= lim_(x -> 3^+) 1/ (x + 1)`
`= lim_(x -> 3^+) 1/ (3 + 1)`
`= 1/4`

Решение

`lim_(x -> (4/3)^-) -x`

`lim_(x -> (4/3)^-) (3x^2 - 4x)/ (3x - 4)`
`lim_(x -> (4/3)^-) (x(3x - 4))/ (3x - 4)`
`lim_(x -> (4/3)^-) x`
`= 4/3`

Бидејки левиот и десниот лимес не се еднакви, лимесот не постои.
