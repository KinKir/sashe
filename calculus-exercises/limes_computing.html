Лимеси - пресметка

Вежба 1. Нека е дадено:
`lim_(x -> a) f(x) = 2; lim_(x -> a) g(x) = -4; lim_(x -> a) h(x) = 0`
да се најде лимесот
a. `lim_(x -> a) [f(x) + 2g(x)]`

Решение

Според Теоремата 2,
`lim_(x -> a) [f(x) + 2g(x)] = lim_(x -> a) f(x) + 2lim_(x -> a) g(x) = 2 + 2*(-4) = -6`

`■`

b. `lim_(x -> a) [h(x) - 3g(x) +1]`
Решение

Според Теоремата 2,
`lim_(x -> a) [h(x) - 3g(x) +1] = lim_(x -> a) h(x) - 3lim_(x -> a) g(x) + 1 = 0 - 3*(-4) + 1 = 13`

`■`

c. `lim_(x -> a) [f(x) * g(x)]`
Решение

Според Теоремата 2,
`lim_(x -> a) [f(x) * g(x)] = lim_(x -> a) f(x) * lim_(x -> a) g(x) = 2 * (-4) = -8`

`■`

d. `lim_(x -> a) root(3) [6 + f(x)]`
Решение

Според Теоремата 2,
`lim_(x -> a) root(3) [6 + f(x)] = root(3) (lim_(x -> a) (6 + f(x)) = root(3) 8 = 2`

`■`

e. `lim_(x -> a) 2/g(x)`
Решение

Според Теоремата 2,
`lim_(x -> a) 2/g(x) = (lim_(x->2) 2)/( lim_(x->2) g(x)) = 2/(-4)= -1/2`

`= 12`

`■`

`= 16`

`= 2`

Пресметување на лимес - кога функцијата се стреми кон определена вредност

Вежба 1. Да се определат лимесите:
а) `lim_(x -> 3) (x^2 - 2x)/(x + 1)`

Решение

`x + 1 !=0` па вредноста на лимесот `f(x) = (p(x))/(q(x)) = f(a)`

`f(a) = (3^2 - 2*3)/(3 +1) = 3/4`

или

`((lim_(x -> 3) x)^2 - lim_(x -> 3)2 * lim_(x -> 3) x )/ (lim_(x -> 3) x + lim_(x -> 3)1)`

`(3^2 - 2 * 3 ) / (3 + 1)`

`= 3/4`

`■`

б) `lim_(x -> 0) (x^2 - 16)/(x - 4)`
Решение

`x - 4 !=0` па вредноста на лимесот `f(x) = (p(x))/(q(x)) = f(a)`

`f(a) = (0^2 - 16)/(0 - 4) = 4`

или

`((lim_(x -> 0) x)^2 - lim_(x -> 0)16)/ (lim_(x -> 0) x -lim_(x -> 0)4)`

`(0^2 - 16) / (0 - 4)`

`= 4`

`■`

Пресметување на лимес - кога вредноста на функцијата е неопределена

Да се определат лимесите:

`lim_(x -> 3) (x^2 - 2x - 3)/(x - 3)`
Решение

Кога `x -> 3` се добива неопределениот облик `0/0`.

Треба да извршиме расчленување

`x^2 - 2x - 3 = (x - x_1)(x - x_2)`

`x_(1,2) = (2 +- sqrt(4 + 12))/2 => x_1 = 3, x_2 = -1`

`x^2 - 2x - 3 = (x - 3)(x + 1)`

`lim_(x -> 3) (x^2 - 2x - 3)/(x - 3) = lim_(x -> -1) ((x - 3)(x + 1))/(x - 3)`

`= lim_(x -> 3)(x + 1)`

`L = 4`

`■`

`lim_(x -> -1) (x^2 - 3x - 4)/(x + 1)`
Решение

Кога `x -> -1^+` се добива неопределениот облик `0/0`.

Треба да извршиме расчленување

`x^2 - 3x - 4 = (x - x_1)(x - x_2)`

`x_(1,2) = (3 +- sqrt(9 + 16))/2 => x_1 = 4, x_2 = -1`

`x^2 - 3x - 4 = (x - 4)(x + 1)`

`lim_(x -> -1) (x^2 - 3x - 4)/(x + 1) = lim_(x -> -1) ((x - 4)(x + 1))/(x + 1)`

`= lim_(x -> -1)(x - 4)`

`L = -5`

`■`

`lim_(x -> 2) (x^2 + 3x - 10)/(x^2 - 8x + 12)`
Решение

Кога `x -> 2` се добива неопределениот облик `0/0`.

Треба да извршиме расчленување на броителот и именителот

`x^2 + 3x - 10 = (x - x_1)(x - x_2)`

`x_(1,2) = (-3 +- sqrt(9 + 40))/2 => x_1 = -5, x_2 = 2`

`x^2 + 3x - 10 = (x + 5)(x - 2)`

`x^2 - 8x + 12 = (x - x_1)(x - x_2)`

`x_(1,2) = (8 +- sqrt(64 - 48))/2 => x_1 = 6, x_2 = 2`

`x^2 - 8x + 12 = (x - 6)(x - 2)`

`lim_(x -> 2) ((x - 2)(x + 5))/((x - 2)(x - 6)) = lim_(x -> -1) (x + 5)/(x - 6)`

`L = -7/4`

`■`

`lim_(x -> -5) (x^2 + 5x)/(x^2 - 25)`
Решение

Кога `x -> =-5` се добива неопределениот облик `0/0`.

Треба да извршиме расчленување на броителот и именителот

`x^2 + 5x = x(x + 5)`

`x^2 - 25 = (x - 5)(x + 5)`

`lim_(x -> -5) x(x + 5)/((x - 5)(x + 5)) = lim_(x -> -1) x/(x - 5)`

`L = 1/2`

`■`

`lim_(x -> 1^+) (x^4 - 1)/(x - 1)`
Решение

Кога `x -> 1^+` вредноста на именителот е `x - 1 =0` па функцијата не е определена

Треба да извршиме расчленување

`lim_(x -> 1^+) ((x^2 -1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) ((x -1)(x + 1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) (x + 1)(x^2 + 1)`

`= 4`

`■`

`lim_(x -> 4) (4 - x)/(2 - sqrt(x))`
Решение

Кога `x -> 1^+` вредноста на именителот е `x - 1 =0` па функцијата не е определена

Треба да извршиме расчленување

`lim_(x -> 1^+) ((x^2 -1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) ((x -1)(x + 1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) (x + 1)(x^2 + 1)`

`= 4`

`■`

`lim_(x -> 0) (1/x - 1/(x^2 +x))`
Решение

Кога `x -> 1^+` вредноста на именителот е `x - 1 =0` па функцијата не е определена

Треба да извршиме расчленување

`lim_(x -> 1^+) ((x^2 -1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) ((x -1)(x + 1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) (x + 1)(x^2 + 1)`

`= 4`

`■`

`lim_(x -> 4) abs(x - 4)`
Решение

Кога `x -> 1^+` вредноста на именителот е `x - 1 =0` па функцијата не е определена

Треба да извршиме расчленување

`lim_(x -> 1^+) ((x^2 -1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) ((x -1)(x + 1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) (x + 1)(x^2 + 1)`

`= 4`

`■`

`lim_(x -> -1) (x^3 + 1)/(x + 1)`
Решение

Кога `x -> 1^+` вредноста на именителот е `x - 1 =0` па функцијата не е определена

Треба да извршиме расчленување

`lim_(x -> 1^+) ((x^2 -1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) ((x -1)(x + 1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) (x + 1)(x^2 + 1)`

`= 4`

`■`

`lim_(x -> 0) (sqrt(4 + x) - 2)/x`
Решение

Кога `x -> 1^+` вредноста на именителот е `x - 1 =0` па функцијата не е определена

Треба да извршиме расчленување

`lim_(x -> 1^+) ((x^2 -1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) ((x -1)(x + 1)(x^2 + 1))/(x - 1)`

`lim_(x -> 1^+) (x + 1)(x^2 + 1)`

`= 4`

`■`

`lim_(t -> 1) (t^3 + t^2 - 5t + 3)/(t^3 - 3t + 2)`
Решение

Кога `t -> 1` добиваме неопределен облик `0/0`

`t^3 -t -2t + 2 = t(t^2 - 1) - 2 (t - 1) = t(t - 1)(t + 1) - 2(t - 1) = (t - 1)(t^2 + t - 2)`

`t^2 + t - 2 = (t - t_1)(t + t_1)`

`t_(1,2) = (-1 +- sqrt(1 + 8))/2 => t_1 = -2, t_2 = 1`

`t^2 + t - 2 = (t - 1)(t + 2)`

`t^3 - t -2t + 2 = (t - 1)(t - 1) (t + 2)`

`t^3 + t^2 -5t + 3 = t^3 + 2t^2 - t^2 - 3t - 2t +3 = t^2(t - 1) + 2t(t - 1) - 3(t - 1) = (t - 1)(t^2 + 2t -3)`

`t^2 + 2t - 3 = (t - t_1)(t - t_2)`

`t_(1.2) = (-2 +- sqrt(4 + 12))/2 => t_1 =-3, t_2 = 1`

`t^2 + 2t - 3 = (t - 1)(t + 3)`

`lim_(t -> 1) (t^3 + t^2 - 5t + 3)/(t^3 - 3t + 2) = lim_(t -> 1) ((t-1)(t-1)(t+3))/((t-1)(t-1)(t+2))`

`lim_(t -> 1) (t+3)/(t+2)`

`= 4/3`

`■`

`lim_(x -> 2) x/(x^2 - 4)`
Решение

Кога `x -> 2` добиваме неопределен облик `2/0`

`lim_(x ->2^-) x/(x^2 -4) = -oo`

`lim_(x ->2^+) x/(x^2 -4) = +oo`

Што треба да се знае на памет
  • Интуитивна метода:
    `lim_(x->a) f(x) = L` ,`f(x) rarr L`   кога   `x rarr a` , `lim_(x->a^+) f(x) = L` , `lim_(x -> a^-) f(x) = L`
    `lim_(x -> a) f(x)` постои само ако `lim_(x -> a^+) f(x) = lim_(x -> a^-) f(x) = L`
  • Вертикални асимптоти - `a` е асимтота на графикот на функцијата `f(x)` ако
    `lim_(x -> a^-) f(x) = +oo`,`lim_(x_> a^+) f(x) = +oo`, `lim(x -> a^-) f(x) = -oo`, `lim_(x -> a^+) f(x) = -oo`
  • Основни лимеси:
    `lim_(x -> a) k = k`,  `lim_(x -> a) x = a`,  `lim_(x -> 0^-) f(1/x) = -oo`, `lim_(x -> 0^+) f(1/x) = +oo`
  • Аритметички операции над лимеси
    `lim_(x -> a) [f(x) + g(x)] = lim_(x -> a) f(x) + lim_(x -> a) g(x)`
    `lim_(x -> a) [f(x) - g(x)] = lim_(x -> a) f(x) - lim_(x -> a) g(x)`
    `lim_(x -> a) [f(x)*g(x)]   = lim_(x -> a) f(x) * lim_(x -> a) g(x)`
    `lim_(x -> a) [f(x) / g(x)]         = (lim_(x -> a) f(x)) /(lim_(x -> a) g(x))` ако `lim_(x -> a) g(x) != 0`
  • Лимес од полиноми и рационални функции
    `lim_(x-> a)p(x) = p(a)`
  • Дробно рационална функција
    `f(x) = (p(x))/(q(x)) => lim_(x -> a) f(x) = f(a), q(a)!= 0`
  • Неопределен облик `0/0`
    а) ако броителот и именителот имаат заеднички множител прво се врши кратење па се пресметува лимес на упростената функција

Кога функцијата се стреми кон одредена вредност за пресметка на лимесот се користи методот на замена, односно.

`lim_(x -> a)f(x) = f(a)`

Ова е прв метод кој треба да се користи при решавање на лимеси.

Кога вредноста на функцијата е неопределена, односно.

`lim_(x -> a)f(x) != f(a)`

треба да се користи методот на факторизација, односно расчленување, на алгебарскиот(те) израз(и).
