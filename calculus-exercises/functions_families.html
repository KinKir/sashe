Фамилии од функции

Вежба 1. Да се определи равенка за:
а) фамилијата прави кои се паралелни со правата `2x + 4y = 1`.

Решение

Функцијата треба да се запише во облик ` y = mx +b`

`y = -1/2x + 1/4`

Ако `m` се фиксира и `b` се третира како параметар, добиваме фамилија од паралелни прави чии членови имаат коефициент на правец `m`. Па според тоа фамилијата н аправи е:

`y = 1/2 x + b`

`■`

б) фамилијата прави кои минуваат низ пресекот на правите `5x - 3y + 11 = 0` и `2x - 9y + 7 = 0`
Решение

Правите се зададени експлицитно `(ax + by + c)` па пресекот се одредува со примена на Крамеровото правило. Прво се означуваат детерминантите.

`Delta = [[a, b],[a_1, b_1]] = [[5, 3],[2, -9]]`

`Delta_x = [[c, b],[c_1, b_1]] = [[11, 3],[7, -9]]`

`Delta_y = [[a, c],[a_1, c_1]] = [[5, 11],[2, 7]]`

Знаеме дека ако `Delta != 0` тогаш постои единствена точка `(x_0, y_0)` на пресек одредена со

`x_0 = Delta_x/Delta; y_0 = Delta/Delta_y`

`Delta = ab_1 - a_1b = 5(-9) - 2*(-3) = -45 + 6 = -39`

`Delta_x = b_1c - c_1b = 11*(-9) - 7*3 = -99 - 21 = 120`

`Delta_y = ac_1 - a_1c = 5*7 - 2*11 = 35 - 22 = 13`

`x_0 = Delta_x/Delta; y_0 = Delta_y/Delta`

`x_0 = 120/-39; y_0 = 120/13`

`5x - 3y + 11 = 0 => -3y = -11 - 5x => y = 11/3 - 5/3x`

`2x - 9y + 7 = 0 => -9y = -7 - 2x => y = -7/9 - 2/9x`

`11/3 - 5/3x = -7/9 - 2/9x`

`-(5/3)x + (2/9)x = -7/9 - 11/3`

`(-15x + 2x)/9 = (-7 -33)/9`

`x = -40/13`

`5x - 3y + 11 = 0 => 5x = 3y - 11 => x = 3/5y -11/5`

`2x - 9y + 7 = 0 => 2x = 9y -7 => x = 9/2y -7/2`

`3/5y -11/5 = 9/2y -7/2`

`3/5y -9/2y = 11/5 -7/2`

`6y -45y/10 = 22 - 14/10`

`39y = 8`

y = 8/39

`■`

в) фамилијата прави кои се тангенти на кружницата со центар во `(0,0)` и радиус `3`.
Решение

Тангентата во точката `(x_0, y_0)` е нормална на правата која минува низ точката `(x_0, y_0)` и центарот `(0, 0)`, бидејќи коефициентот на оваа права е `(y_0 - 0)/(x_0 - 0)` следува дека коефициентот на тангентата е `k = -1/(y_0/x_0) = x_0/y_0`. Оттука равенката на тангентата во точката `(x_0, y_0 )` е `y - y_0 = x_0/y_0 (x-x_0) => y = -(x_0^2 + y_0^2 -x_0x)/y_0`

Бидејќи точката `(x_0, y_0)` лежи на кружницата `x_0^2 +y_0^2 = 3^2` т.ш. равенката на бараната фамилија прави е

`y = +_ (9- x_0x)/(sqrt(9 - x_0^2)`

Вежба 2. За секој график да се најде соодветната равенка од следните
а) `y = root(5)x`; б) `y = 2x^5`; в) `y = -1/x^8`;  г) `y = sqrt(x^2 - 1)`; д) `root(4)(x - 2)`;  ѓ) `y = -root(5)(x^2)`

Решение

`I - д`, `II - ѓ`, `III - в`, `IV - б`, `V - г`, `VI - а`

`■`

Што треба да се знае на памет
  • Равенка на права низ две точки:
    `y - y_1 = (y_2 - y_1)/(x_2 - x_1) (x - x_1)`
  • Коефициент на правец:
    `m = (y_2 - y_1)/(x_2 - x_1)`
  • Равенка на права со даден коефициент на правец
    `y - y_1 = m(x - x_1)`
  • Дискусија за коефициентот на правец:
    Ако `m > 0` правата расте од лево на десно
    Ако `m < 0` правата опаѓа од лево на десно
    Ако `m = 0` правата е хоризонтала
  • Експлицитен облик на равенка на права:
    `y = mx + b`
  • Паралелни и нормални прави:
    `m_1 = m_2`  паралелни прави
    `m_1m_2 = -1`  нормални први
