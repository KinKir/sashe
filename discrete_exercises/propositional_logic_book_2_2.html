Вежба 1. Нека `A` е множеството на студенти кои живеат на помалку од два километри од факултетот и нека `B` е множеството на студентите кои одат пешки на часови. Да се опуишат студентите во секое од овие множества.
a) `A cap B`

Решение

`A cap B = {`само студенти кои живееат на помалку од 2 километри и истовремено одат пешки на факултет`}`

b) `A cup B`
Решение

`A cuap B = {`од сите студенти кои живееат на помалку од 2 километри и сите кои одат пешки на факултет`}`

c) `A - B`
Решение

`A - B = {`студенти кои живееат на помалку од 2 километри и не одат пешки на факултет`}`

d) `B - A`
Решение

`B - A = {`студенти кои одат пешки на факултет но не живеат на помалку од 2 километри`}`

Вежба 5. Да се докаже законот закомплемент подажувајки дека `(A^C)^C =A`.

Решение

Според дефиниција `(A^C)^C` е множество од елементи кои не се наоѓаат во множеството `A^C`. Да не се наоѓа во `A^C` значи да се биде во `A`. Значи `(A^C)^C` е истото множество како и `A`.
`(A^C)^C = {x | not x in A^C} = {x | not not x in A} = {x | x in A} = A`

Вежба 6. Да се докаже законот за идентитет покажувајки дека:
i) `A cup emptyset = A`

Решение

Според дефиниција за унија на две множества `A cup B` е множество од елементите кои се наоѓаат или во едното или во другото множество.

`A cup emptyset = {x | x in A vv x in emptyset} = { x | x in A vv x in _|_} = {x | x in A} = A`

ii) `A cap U`
Решение

Според дефиниција за пресек на две множества `A cup B` е множество од елементите кои се наоѓаат и во едното и во другото множество.

`A cap U = {x | x in A cup U} = { x | x in A ^^ x in U}`

Бидејки секој елемент е во универзалното множество `U` следе дека `x in U = TT`.

`A cap U = {x | x in A cup U} = { x | x in A ^^ x in U} = {x | x in A ^^ TT}`

Според законот за идентитет за искази, следи

`A cap U = {x | x in A cup U} = { x | x in A ^^ x in U} = {x | x in A ^^ TT} = {x | x in A} = A `

Вежба 7. Да се докаже законот за доминација покажувајки дека:
i) `A cup U = U`

Решение

`A cup U = {x | x in A cup U} = { x | x in A ^^ x in U} = {x | x in A ^^ TT}`

Според законот за доминација за искази, следи

`A cup U = {x | x in A cup U} = { x | x in A ^^ x in U} = {x | x in A ^^ TT} = {x | TT} = U`

ii) `A cap emptyset = emptyset`
Решение

`A cap emptyset = {x | x in A cap emptyset} = { x | x in A ^^ x in emptyset} = {x | x in A ^^ _|_}`

Според законот за доминација за искази, следи

`A cap emptyset = {x | _|_} = { x | x in emptyset} = emptyset`

Вежба 8 . Да се докаже законот за доминација покажувајки дека:
i) `A cup A = A`

Решение

`A cup A = {x | x in A cup x in A} = {x | x in A vv x in A}`

Според законот за идемпотентност за искази (`p vv p = p; p ^^ p = p), следи

`A cup A = {x | x in A} = A `

ii) `A cap A = A`
Решение

`A cap A = {x | x in A cap x in A} = {x | x in A AA x in A}`

Според законот за идемпотентност за искази (`p vv p = p; p ^^ p = p), следи

`A cup A = {x | x in A} = A `

Вежба 9 . Да се докаже законот за комплемент покажувајки дека:.
i) `A cup (A)^C = U`

Решение

`A cup (A)^C = {x | x in A vv x in (A)^C} = {x | x in A vv x not in A} = {x | x in A vv U} = U`

ii) `A cap (A)^C = emptyset`
Решение

`A cap (A)^C = {x | x in A ^^ x in (A)^C} = {x | x in A ^^ x not in A} = {x | _|_} = emptyset`

Вежба 10 . Покажи дека:
i) `A - emptyset = A`

Решение

`A - emptyset = {x | x in A - emptyset} = {x | x in A ^^ notin emptyset} = {x | x in A} = A`

ii) `emptyset - A = emptyset`
Решение

`A - emptyset = {x | x in emptyset - x in A } = {x | x in emptyset ^^ notin A} = {x | x in emptyset} = emptyset`

Вежба 18 . Нека `A`, `B` и `C` се множества. Да се докаже дека:
i) `(B - A) cup (C - A) = (B cup C) -A`

Решение

Нека `x in (B - A) cup (C - A)`

Според дефиниција за унија:

`x in (B - A) vv x in (C - A)`

Според дефиниција за разлика на множества:

`(x in B ^^ not x in A) vv (x in C ^^ not x in A)`

Според диструбутивен закон за искази:

`(x in B vv x in C) ^^ not (x in A)`

Според дефиниција за унија:

`(x in B cup C) ^^ not (x in A))`

Според дефиниција за разлика:

`x in (B cup C) - A`

Според дефиниција за подмножество:

`(B - A) cup (C - A) subset (B cup C) - A`

---------------------------

Нека `x in (B cup C) - A`

Според дефиниција за разлика:

`x in (B cup C) ^^ not x in A`

Според дефиниција за унија:

`(x in B vv x in C) ^^ not x in A`

Според диструбутивен закон за искази:

`(x in B ^^ not (x in A)) vv ( x in C ^^ not (x in A))`

Според дефиниција за разлика:

` x in B - A vv x in C - A `

Според дефиниција за унија:

`x in (B - A) cup (C - A ) ` +

Според дефиниција за подмножество:

`(B cup C) - A subset (B - A) cup (C - A )`

Бидејки:

`(B - A) cup (C - A) subset (B cup C) - A` и ѝ(B cup C) - A subset (B - A) cup (C - A )` - двете множества се исти.
