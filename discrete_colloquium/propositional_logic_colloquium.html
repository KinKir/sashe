Задача 3.
а) filter_7 Со користење на логички еквивалнтни формули покажете дека следната формула е тафтологија.
`((not p vv q) -> not r) -> (r -> p)`

Решение

`((not p vv q) -> not r) -> (r -> p) -= not((not p vv q) -> not r)) vv (r -> p)`
`not((not p vv q) -> not r)) vv (r -> p) -= not(not(not p vv q) vv not r)) vv (not r vv p)`
`not(not(not p vv q) vv not r)) vv (not r vv p) -= not((p ^^ not q) vv not r)) vv (not r vv p)`
`not((p vv not q) vv not r)) vv (not r vv p) -= not(p ^^ not q) ^^ r vv (not r vv p)`
`not((p ^^ q) vv not r) vv (not r vv p) -= not p vv q ^^ r vv not r vv p`
`(not(p ^^ q) ^^ r) vv (not r vv p) -= not p vv p vv q ^^ r vv not r`
`not p vv p vv q ^^ r vv not r -= TT vv q ^^ TT`
`TT vv q -= TT vv q ^^ TT`

filter_3Задача 4 (3+2+1)- 19.11.2018 I Група. За реченицата „Има бербер кој ги бричи сите мажи во градот кои не се бричат сами“ да се дефинираат соодветните предикати, со нив и со користење на симболите и сврзниците од исказното и предиктивното сметање да се запише дадената реченица, да се направи нејзина негација и негацијата да се запише во говорниот јазик.

Решение

`EEx(man(x) ^^ AAy(man(y) -> (shaves(x, y) harr not shaves(y, y))))`
This sentence is unsatisfiable (a contradiction) because of the universal quantifier `AA`. The universal quantifier y will include every single element in the domain, including our infamous barber x. So when the value x is assigned to y, the sentence can be rewritten to `shaves(x, y) harr not shaves(x, x)`, which is an instance of the contradiction `a harr not a`.
