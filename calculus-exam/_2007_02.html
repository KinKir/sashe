Задача 1

4 За функцијата `f(x) = (sqrt(x) - 2)/(x - 4)` да се определи функцијата со идентичен график, но без прекин.

Решение

Функцијата `f(x) = (sqrt(x) - 2)/(x - 4)` има прекин точката `x = 4`. Со разложување на функцијата `f(x)` се добива функија `g(x)` која има ист график но друг домен.

`f(x) = (sqrt(x) - 2)/(x - 4) = (sqrt(x) - 2)/((sqrt(x))^2 -2^2`
`= (sqrt(x) - 4)/((sqrt(x) - 2)(sqrt(x) + 2))`
`= 1 / (sqrt(x) + 2)`

[5] Да се пресмета `lim_(x->+oo)(sqrt(x^2 + ax) - sqrt(x^2 + bx))`.

Решение

Чекор 1. Броителот и именителот ги множиме со конјугација на изразот во броителот
`lim_(x->+oo)(sqrt(x^2 + ax) - sqrt(x^2 + bx))`
`= lim_(x->+oo)(sqrt(x^2 + ax) - sqrt(x^2 + bx)) * (sqrt(x^2 + ax) + sqrt(x^2 + bx))/(sqrt(x^2 + ax) + sqrt(x^2 + bx))`
`= lim_(x->+oo)((x^2 + ax) - (x^2 + bx))/(sqrt(x^2 + ax) + sqrt(x^2 + bx)) = (ax - bx)/(sqrt(x^2 + ax) + sqrt(x^2 + bx))`

Делиме со најголемиот степен во именителот `x`.
`= lim_(x->+oo)(a - b)/(sqrt(1 + (a/x)) + sqrt(1 + (b/x))`

Сега со замена добиваме:
`= lim_(x->+oo)(a - b)/(sqrt(1 + (a/(oo))) + sqrt(1 + (b/(oo))`
` = (a-b)/2`

Задача 5

12 Да се анализира и скицира графикот на функцијата:
`f(x) = lnx/sqrt(x)`

Решение

Чекор 1: Домен на функцијата.

Нека `g(x) = lnx`. Домен на функцијата `g(x)` e `D_g= [0, +oo)` и нека `h(x) = 4x^2`. Доменот на функцијата `h(x)` е `D_(4x^2) = (-oo, +oo)`. Домен на функцијата `f(x) = g(x)/h(x) = D_f = D_g nn D_h \ {x in D_g | h(x) = 0}`
`D_f = [0, +oo) nn (-oo, 0] uu [0, +oo)= [0, +oo)`
